# Real-Time Options Pricing & Risk Analysis System

An integrated financial system that automatically fetches real market data, calculates option prices using both analytical (Black-Scholes) and numerical (Monte Carlo) methods, and provides comprehensive risk analysis.

## Features

- 🔄 **Auto-fetches market data** - Just enter a ticker symbol
- 📊 **Multiple volatility estimators** - Close-to-Close, Parkinson, EWMA, Garman-Klass
- 📐 **Black-Scholes pricing** - Analytical model with all Greeks
- 🎲 **Monte Carlo simulation** - Numerical model with confidence intervals
- ✅ **Model validation** - Compares both methods for consistency
- 📈 **Comprehensive risk metrics** - Greeks, probabilities, breakeven analysis
- 📊 **Visual dashboard** - 10+ charts showing all aspects
